COMPUTER

Monitor:

An electronic output device which displays images in pictorial form on the screen.

CPU:

The CPU is the brain of a computer,containing all the circuitry needed to process input store data , and output results.

Keyboard:

It consists of multiple buttons,which includes numbers,special keys like the windows and alt key,including performing other functons.

Mouse:

A mouse is a small device that a computer user pushes across a desk surface in order to point to a place on a display screen and to select one or more actions to take from that position.
